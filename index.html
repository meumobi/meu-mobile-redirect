meu-mobile-redirect: Helper

To simulate a mobile browsing check out User-Agent String of mobile browser, for example iOS7

If you need a "basic" Redirect, ie all requests to website from mobile should be redirect to an unique mobile url, then only set mmrConfig.mobileUrl field.
